About this

This project is to show an example of raymarching and showing how it can be used in a 3D context. Project inspired by a friend's project.

W,A,S,D to move, mouse is to look.

If it doesn't seem to work, open the console and type "player.keys" and hit enter, then close the console. It doesn't work sometimes and I'm not too sure why.
You can also move the pillars by clicking on them.

Made by jaideng1.
